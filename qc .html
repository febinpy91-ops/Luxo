One-time setup — do this once
1Open console.firebase.google.com in Safari, sign in with Google, tap Create a project (name it "formave"). Skip Analytics.
2Inside the project, tap the </> (web) icon to add a web app. Give it any nickname, tap Register.
3It shows a firebaseConfig block. Copy everything inside the { } and paste it in the box above.
4In the left menu open Build → Firestore Database → Create database. Choose a location, then Start in test mode and Enable.
5On the editor's phone: enter your name, pick EDITOR, paste the key, tap Connect. On each partner's phone: pick PARTNER, paste the same key, tap Connect.
Tip: this is easiest done once on a computer, then partners just paste the same key on their phones. Only one device should be the EDITOR.
